### What I Build

I specialize in the full ML delivery lifecycle — from prototype to production:

- **Agentic AI Systems** — LangGraph orchestration, multi-agent pipelines, RAG architectures with ChromaDB and LangSmith evaluation
- **MLOps & CI/CD** — MLflow tracking, automated model monitoring (Evidently, Arize Phoenix, Azure ML Monitor), GitHub Actions pipelines, CT/CD for ML
- **Healthcare AI** — HL7/FHIR integration (Mirth Connect certified), clinical risk prediction (MySurgeryRisk, AUC 0.82–0.94), HIPAA-compliant data pipelines
- **Computer Vision** — YOLOv8 + ByteTrack for real-time object detection and tracking (surgical robotics, warehouse systems)
- **Cloud & Infrastructure** — Azure ML, AWS SageMaker, Databricks, PySpark, GCP
